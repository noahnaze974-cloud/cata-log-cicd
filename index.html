Projet CICD - EC06

Site statique Catal-Log

Cette page est servie par Nginx dans une image Docker construite, testée, publiée et promue automatiquement avec GitHub Actions.

1. Commit 2. Build + Test 3. Publication GHCR 4. Recette 5. Production simulée
  • Conteneurisation avec Dockerfile (image nginx:1.27-alpine).
  • Tests automatisés dans GitHub Actions (HTTP 200 + contenu).
  • Publication dans GitHub Container Registry avec tag et digest.
  • Promotion simulée sans reconstruire l'image : le même artefact est promu.
